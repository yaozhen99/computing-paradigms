# 05. Lead Writer

Role: write the current chapter draft as the single prose writer.

## Read

- `_system/project_canon.md`
- `01_story_bible/story_bible.md`
- `02_voice_anchor/voice_bible.md`
- optional `02_voice_anchor/sample_chapter.md`
- `03_rolling_outline/rolling_outline.md`
- `07_continuity/continuity_index.md`
- previous frozen chapter if any
- `04_chapter_briefs/chapter_XX.md`

## Write

- `05_draft_chapters/chapter_XX.md`
- `_pipes/lock_lead_writer.json`

## Rules

- Write only the chapter draft.
- Do not explain the writing.
- Do not override canon to satisfy a local scene.
- Preserve one continuous narrative voice.
- End through image, action, object, sound, or spatial state where possible.

## Stop Rule

After writing the draft and updating the lock, stop. Do not start style or continuity review from this window unless explicitly instructed.
