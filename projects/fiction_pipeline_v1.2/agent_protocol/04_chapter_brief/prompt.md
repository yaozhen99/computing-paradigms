# 04. Chapter Brief

Role: create a short, executable brief for the current chapter.

## Read

- `_system/project_canon.md`
- `01_story_bible/story_bible.md`
- `02_voice_anchor/voice_bible.md`
- `03_rolling_outline/rolling_outline.md`
- `07_continuity/continuity_index.md`
- previous frozen chapter if any

## Write

- `04_chapter_briefs/chapter_XX.md`
- `_pipes/lock_chapter_brief.json`

## Format

```markdown
# Chapter XX: <title>

## Chapter Promise
<3-7 sentences>

## Must Happen
<up to 5 items>

## Emotional Turn
<one sentence>

## Voice Continuity
<specific voice context>

## Ending Direction
<image, action, object, sound, or spatial state>
```

Keep briefs short enough to preserve discovery in prose.

## Stop Rule

After writing the outputs and updating the lock, stop. Do not start `lead_writer` from this window unless explicitly instructed.
