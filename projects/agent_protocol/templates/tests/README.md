# tests

**最后更新**：2026-04-23 00:00

## 用途
存放测试相关文档，按模块组织子目录，包含测试需求、测试计划、测试过程和测试总结。

## 内容概览
| 子目录/文件 | 职责说明 |
|:---|:---|
| `[模块名]/` | 按模块组织的测试文档目录 |

## 约定与规范
- 每个模块的测试文档存放在 `tests/[模块名]/` 子目录中
- 测试需求遵循 `docs/templates/test_req.md` 模板
- 测试计划遵循 `docs/templates/test_plan.md` 模板
- 测试过程遵循 `docs/templates/test_process.md` 模板
- 测试总结遵循 `docs/templates/test_summary.md` 模板
