# proposals

**最后更新**：2026-04-23 00:00

## 用途
存放协议修正建议文件，配合 AGENTS.md 第 6.4 条款使用。AI 助手认为协议需要修正时，必须将建议生成为独立文件存放于此，**严禁直接修改 AGENTS.md 或模板文件**。

## 内容概览
| 子目录/文件 | 职责说明 |
|:---|:---|
| （暂无文件） | 协议修正建议将在 AI 检测到协议缺陷时按需生成 |

## 约定与规范
- 文件命名必须遵循 `protocol_change_[YYYY-MM-DD].md` 格式
- 文件内容必须包含：建议原因、涉及条款、具体修改建议、预期效果
- 修改权仅属于项目管理者（人），AI 仅拥有建议权
