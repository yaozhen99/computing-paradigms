## 需求描述

### 核心目标

开发一个本地文件加密命令行工具。用户可以通过简单命令对指定文件进行加密（lock）和解密（unlock），加密后的文件以 .vault 后缀存储，密钥由用户口令派生。

### 功能范围

1. `vault init` — 初始化本地密钥库（生成 .vault 目录，存储盐值和配置）
2. `vault lock <file>` — 加密指定文件，生成同名 .vault 文件，原文件可选保留或删除
3. `vault unlock <file.vault>` — 解密 .vault 文件，还原原始文件
4. `vault list` — 列出当前目录下所有 .vault 加密文件及其状态
5. 加密算法：AES-256-GCM，密钥派生：PBKDF2-SHA256
6. 口令输入不可回显，从终端安全读取

### 不做什么

- 不做网络传输、云存储、远程密钥管理
- 不做文件夹递归加密（v1 只处理单文件）
- 不做图形界面
- 不做多用户/权限系统
