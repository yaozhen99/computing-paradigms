# skills

**最后更新**：2026-04-23 00:00

## 用途
存放技能文档，包含项目专属技能和通用技能模式，支持 AI 自我演化的知识沉淀。

## 内容概览
| 子目录/文件 | 职责说明 |
|:---|:---|
| `project_specific.md` | 项目专属技能库，记录与本项目特定架构、业务规则、模块依赖相关的经验 |
| `general_patterns.md` | 通用技能与模式库，记录与具体项目无关、可通用的设计模式、算法实现、调试技巧 |

## 约定与规范
- `project_specific.md` 内容结构遵循 `docs/templates/project_skill.md` 模板
- `general_patterns.md` 内容结构遵循 `docs/templates/general_skill.md` 模板
- 每次任务完成后必须评估是否需要更新技能文档
- 若无新增知识，必须在技能文档末尾追加 `## [YYYY-MM-DD] 本次任务无新增通用技能`
