# 世界观构建指南

## 概述

世界观是科幻小说的根基。没有自洽的世界观，后续一切创作都建立在沙上。

## 必须覆盖的维度

### 1. 物理法则

- 本世界的物理法则与现实有何不同？
- 这种差异的底层原理是什么？
- 差异带来的连锁反应是否自洽？

### 2. 技术树

- 核心技术是什么？原理是什么？
- 技术的发展阶段（萌芽 / 成熟 / 过剩）？
- 技术的社会影响（谁受益、谁受害、谁控制）？
- 技术的边界和限制（不能做什么，为什么）？

### 3. 社会结构

- 政治体制（集权 / 联邦 / 无政府 / AI 治理）？
- 阶层划分的依据（财富 / 基因 / 意识等级 / 技术接入权）？
- 权力运作方式（明规则 / 潜规则）？
- 边缘群体如何生存？

### 4. 历史线

- 至少 3 个关键历史事件
- 每个事件的因果链
- 历史如何塑造了当前的世界格局

### 5. 日常生活

- 普通人的一天是什么样的？
- 衣食住行有什么不同？
- 文化娱乐有什么变化？

## 自洽性检查清单

- [ ] 技术树与社会结构是否匹配？（高技术不应出现低治理）
- [ ] 物理法则的例外是否被充分解释？
- [ ] 历史事件的因果链是否完整？
- [ ] 日常生活的细节是否与设定一致？
- [ ] 是否存在"万能解"设定？（必须有限制）

## 常见陷阱

- **技术万能症** — 一项技术解决所有问题，没有代价
- **历史真空** — 世界仿佛凭空出现，没有演变过程
- **社会扁平化** — 所有人对技术/事件的反应一致
- **设定堆砌** — 罗列设定但不建立内在联系
