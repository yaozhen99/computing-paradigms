# Canon Room

Purpose:

Hold durable truth and continuity facts.

Allowed material:

- approved canon
- timelines
- character states
- location states
- form states
- object states
- continuity ledger entries

Forbidden material:

- raw notebook fragments
- unapproved ammo
- contradictory variants without conflict marking
- publication freezing decisions

Default roles:

- `Canon Keeper`
- `Chief Author`
- `Continuity Editor`

Default flow:

`author approval -> canon_room -> chapter packets / editorial checks`
