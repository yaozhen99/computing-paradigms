# Roles

Roles describe what a window is supposed to do.

## Core roles

- `Chief Author`
- `Idea Assistant`
- `Research Assistant`
- `Ammo Librarian`
- `Canon Keeper`
- `Development Editor`
- `Continuity Editor`
- `Line Editor`
- `Copy Editor`
- `Proofreader`
- `Publisher`

## Rule

A role is not a room.  
A role is not a model.  
A role is the window's working identity inside a room.

## Required fields

- role name
- purpose
- allowed actions
- disallowed actions
- default room
- default parent context
