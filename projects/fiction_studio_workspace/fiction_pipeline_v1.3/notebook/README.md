# Notebook

Purpose:

Low-friction capture for the author's raw material.

Allowed material:

- sentence fragments
- dreams
- images
- dialogue tones
- place names
- species behavior
- scene mechanisms
- ending gestures
- conflict seeds

Forbidden material:

- final canon claims
- frozen prose
- publication decisions

Default roles:

- `Chief Author`
- `Idea Assistant`

Default flow:

`notebook -> ammo_bank`
