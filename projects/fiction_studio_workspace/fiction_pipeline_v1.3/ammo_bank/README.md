# Ammo Bank

Purpose:

Store abundant creative ammunition.

Allowed material:

- raw ideas
- contradictory variants
- high-divergence possibilities
- scene seeds
- sensory images
- character pressure tests
- rejected but still interesting material

Forbidden material:

- automatic canon
- frozen chapter text
- unlogged publication decisions

Default roles:

- `Idea Assistant`
- `Chief Author`
- `Research Assistant`

Default flow:

`notebook -> ammo_bank -> asset_room / author_room / canon_room candidates`
