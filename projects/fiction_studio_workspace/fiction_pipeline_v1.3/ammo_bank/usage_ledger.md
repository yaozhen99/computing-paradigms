# Ammo Usage Ledger

Tracks which ammo has been used, where, how deeply, and whether it can recur.

Each usage entry should include:

- ammo id
- chapter or section
- use type
- depth
- canon impact
- reuse status
- notes

Depth values:

- `glance`
- `minor`
- `scene_mechanism`
- `major_setpiece`
- `structural`

Reuse status values:

- `available`
- `must_change_angle`
- `cooldown`
- `spent`
- `recurring`
- `retired`
