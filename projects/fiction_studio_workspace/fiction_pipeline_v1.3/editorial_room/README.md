# Editorial Room

Purpose:

Review work for publishability.

Editors do not own the final prose by default.

Default roles:

- `Development Editor`
- `Continuity Editor`
- `Line Editor`
- `Copy Editor`
- `Proofreader`

Issue severity:

- `BLOCKER`
- `MAJOR`
- `MINOR`
- `TASTE`

Default flow:

`draft/revision -> editorial_room reports -> Chief Author decision`
