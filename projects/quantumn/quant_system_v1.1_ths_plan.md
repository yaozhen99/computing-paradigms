# quant_system_v1.1_ths — 同花顺实盘量化交易系统

> Hermes H家族第一个正式项目
> 创建日期：2026-05-02
> 项目经理：Tony

## 项目目标

基于现有 quant_system_v0.8_ths（模拟交易版），开发 v1.1 实盘交易版本，在虚拟机环境下运行同花顺客户端，实现可实盘交易的量化系统。

## 与 v0.8 的核心差异

| 维度 | v0.8（模拟） | v1.1（实盘） |
|------|-------------|-------------|
| 交易模式 | 模拟交易 | 实盘交易（默认关闭，显式 --live 开启） |
| 运行环境 | 物理机 | 虚拟机（需适配分辨率/窗口/网络） |
| 安全机制 | 无 | 实盘确认token、操作审计、风控熔断 |
| 成交确认 | 无（SUBMITTED即终态） | 轮询委托状态，确认成交/失败 |
| 工具接口 | 无 | JSON CLI，AI可调用 |
| 通知 | 无 | 飞书通知（下单/成交/风控触发） |

## 关键约束

1. **实盘默认关闭**：不加 --live 参数，所有交易操作走dry-run
2. **AI不自主实盘**：AI可读取状态、生成建议、执行dry-run，实盘执行需人工确认
3. **虚拟机环境**：需适配虚拟机的分辨率、网络延迟、窗口渲染差异
4. **保留v0.8不动**：v1.1是独立目录，不修改v0.8代码
5. **文档先行**：每个阶段产出文档，按AGENTS.md协议执行
