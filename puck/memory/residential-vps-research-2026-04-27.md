# Residential IP VPS 调研报告

**日期**: 2026-04-27
**需求**: 1G-2G内存, 流量≥1T(最好2T+), 发达国家家庭IP

---

## ⚠️ 核心发现

**Residential IP VPS 和普通VPS是完全不同的市场。** 家庭IP资源稀缺且昂贵，市面上几乎没有"大流量+低价格"的Residential IP VPS方案。原因：

1. **家庭IP本质是ISP分配给住宅用户的**，获取渠道有限（ISP合作/P2P网络/移动设备共享）
2. **流量成本极高** — 住宅IP按GB计费，不是按月不限流量
3. **真正的Residential VPS供应商极少**，多数只提供代理(Proxy)而非完整VPS

## 五、关键结论

| 维度 | 结论 |
|------|------|
| **非美国家住宅VPS** | 几乎不存在现成产品 |
| **大流量住宅IP** | 按GB计费的话1TB要$1750+，不可行 |
| **可行路径** | 便宜VPS + ISP静态住宅代理（不限流量） |

## 六、注意事项

1. **Bright Data需要KYC** — 需要视频通话验证
2. **ISP Proxy是SOCKS5/HTTP代理** — 需要在VPS上配置代理转发
3. **BuyVM经常缺货** — 可能需要等补货
4. **RackNerd是年付特价** — 日常价更高
5. **住宅IP的"纯净度"** — ISP Proxy > Residential Proxy，因为ISP Proxy是ISP直接分配的
