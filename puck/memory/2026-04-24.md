### 开发管理工作流程

1. 接收任务 - 确认内容、范围、交付物
2. **建立工作空间** - 创建目录，生成agent_protocol工具包（外骨骼）
3. 初始化开发智能体 - 读AGENTS.md、claude.md、工具包
4. 分配任务 - sessions_spawn派给Claude Code
5. 跟踪进程 - 监控状态、记录问题
6. 搜集确认 - 收集代码、文档，验证完整性
7. 确认入库 - 检查归档位置、目录结构
8. 汇报状态 - 向Atlas汇报

**关键点**：工作空间和外骨骼必须在派活前准备好
