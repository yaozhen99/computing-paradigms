## 1. 设计模式实践
### 后台线程处理长任务
- **场景**：HTTP 请求需要触发长运行任务，但不能阻塞响应
- **代码示例**：
```python
t = threading.Thread(target=run_hermes, args=(task_id, query), daemon=True)
t.start()
self.send_response(202)  # 立即返回 Accepted
```
- **注意事项**：守护线程会在主程序退出时自动终止，不适合需要保证完成的任务

## 2. 算法/数据结构技巧
### 正则清理 ANSI 转义序列
- **问题描述**：终端输出包含颜色代码，需要清理
- **实现要点**：`re.sub(r'\x1b\[[0-9;]*m', '', raw_output)`
- **复杂度**：O(n)

## 3. 调试与问题排查
### WSL 命令执行失败
- **现象**：subprocess 调用 WSL 命令返回非零退出码
- **定位方法**：同时捕获 stdout 和 stderr，合并输出分析
- **根因**：通常是环境变量或路径问题
- **通用解决方案**：
  1. 在 WSL 内手动执行命令验证
  2. 检查虚拟环境激活
  3. 使用绝对路径

## 4. 命令行与工具链
- `wsl -d <distro> -e sh -c "<cmd>"` — 在指定 WSL 发行版中执行命令
- `source ~/venv/bin/activate && <cmd>` — 激活虚拟环境后执行
